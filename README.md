# ガチャアプリ

アプリ起動時のスクリーンショットです

<img width="200" height="400" alt="1000008141" src="https://github.com/user-attachments/assets/5669dc9b-f883-4f03-9c59-aa7b949d5c38" />


* **レイアウト画面はapp/src/main/res/layout フォルダに入れております**
* activity_main.xml
* activity_sub.xml
* activity_third.xml
<br>

* **プログラムはapp/src/main/java/com/example/gachaapp フォルダに入れております**
* AL.java
* MainActivity.java
* SubActivity.java
* ThirdActivity.java
<br>

## 特徴・こだわり
* **ガチャ演出** ：私の好きなアニメ、頭文字Dの演出を付けております
* **モンスターの数** ：60種類以上のモンスターの画像を取り込みました

## 使用技術
* **使用言語** ：Java
* **android version** ：14
* **開発ツール** ：Android Stadio

## アプリの使い方
1.アプリを起動します  
2.「1回引く」か「2回引く」ボタンをタップします(ポイントを消費してガチャを引く)  
3.排出結果が画面に表示されます  
4.「ガチャ選択画面に戻る」ボタンで前の画面にもどります
